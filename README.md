# expo-sdk-example

Minimal [Expo](https://docs.expo.dev/) application showcasing IndoorAtlas positioning using the [IndoorAtlas React Native plugin](https://github.com/IndoorAtlas/cordova-plugin/tree/react-native).

## Requirements

Detailed requirements TBA. Requires at least NodeJS 18+.

## Running development build

Install dependencies with `npm install`. Run on Android with `npm run android`. More detailed instructions (including iOS) coming soon.

Note that running proper IndoorAtlas positioning requieres a physical device.
